# Vibe Camp - Learn to Build with AI 🚀

Vibe Camp is an interactive learning platform designed to teach web development fundamentals and advanced concepts through hands-on lessons. Students learn by building real projects with AI assistance, progressing from basic HTML/CSS to building interactive applications.

## 🎓 Course Structure

Vibe Camp features 4 comprehensive lessons, each building upon previous knowledge:

### Lesson 1: Web Basics (5 Pages)
- **Introduction**: Get started with web development fundamentals
- **Setup**: Configure your development environment
- **HTML**: Learn HTML structure and semantic markup
- **CSS**: Master styling and layout techniques
- **Completion**: Celebrate your first milestone!

### Lesson 2: Build Your Profile Website (3 Pages)
- **Vibe Coding**: Learn modern coding practices with AI
- **Build**: Create a professional profile website
- **Deploy**: Launch your site to the internet

### Lesson 3: Tic Tac Toe Game (5 Pages)
- **Introduction**: Understand game development concepts
- **AI Prompt**: Learn to leverage AI for coding assistance
- **Logic**: Implement game mechanics and state management
- **Personalization**: Customize your game with unique features
- **Deploy**: Share your game with the world

### Lesson 4: Chat Application (6 Pages)
- **Introduction**: Introduction to interactive applications
- **API Setup**: Configure API integrations
- **AI Prompts**: Master advanced AI prompting techniques
- **User Interaction**: Build real-time chat features
- **Upgrades**: Enhance your application with new capabilities
- **Deploy**: Launch your chat app

## 🛠 Tech Stack

- **React 18.3**: Modern UI library with hooks and concurrent features
- **TypeScript**: Type-safe JavaScript for better code quality
- **Tailwind CSS**: Utility-first CSS framework for rapid styling
- **Framer Motion**: Smooth animations and page transitions
- **shadcn/ui**: Beautiful, accessible UI components
- **React Router**: Client-side routing for multi-page navigation
- **Vite**: Lightning-fast build tool and dev server

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ or Bun
- A code editor (VS Code recommended)
- A modern web browser

### Installation

git clone https://github.com/husky-696/vibe-camp.git
cd vibe-camp
npm install

### Running the Application

npm run dev

Navigate to http://localhost:5173 in your browser to start learning!
