# Vibe Camp

Welcome to Vibe Camp, an innovative platform designed to enhance your experience with community-driven activities and events. This documentation provides a comprehensive overview of the project, its features, and how to get involved.

## Overview
Vibe Camp is built around the concept of bringing people together for shared interests. The platform enables users to create, discover, and join events within their local communities.

## Features
- **Event Creation**: Users can create events and invite others to join.
- **Event Discovery**: Find events based on interests, location, and availability.
- **User Profiles**: Create profiles to showcase interests and past events attended.
- **Community Interaction**: Engage with other users through comments and direct messaging.

## Getting Started
To get started with Vibe Camp, clone the repository and install the necessary dependencies:
```bash
git clone https://github.com/husky-696/vibe-camp.git
cd vibe-camp
npm install
```

## Usage
Once installed, you can run the application locally:
```bash
npm start
```
Access the application in your browser at `http://localhost:3000`.
